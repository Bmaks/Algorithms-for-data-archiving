Папка Finished, містить програму написану на Microsoft Visual 2010 C++.
Запускаємо виконуваний файл. В папці з .exe файлом повині міститися створений файл, дані якого будуть архівуватися, і файл, в який будуть записуватися дані архівування. В випадку відсутності файлів, програма закриється.

Бажано записувати приклад великими буквами, або англійськими літерами.

Наприклад, я хочу заархівувати файл методом ЛЗ, тоді у папці з екзе файлом( якщо не бажаєте вказувати повний шлях в програмі ) повинні міститися два файли:

Наприклад:
Kursova.exe
1.txt
2.txt

Активуємо програму, вибираємо перший пункт(тиснемо одиинцю + ентер), далі також перший пункт, вводимо, у нашому випадку: "1.txt" + ентер
"2.txt" + enter. Завершуємо або продовжуэмо роботу. Навіть до заврешненя самої програми, файли уже будуть записані, закриті та збережені.
